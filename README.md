# ⌨️ Typing Speed Test

A simple and modern **Typing Speed Test web app** built using HTML, CSS, and JavaScript.

---

## 🚀 Features

- ⏱ 60-second timer
- 📊 WPM, CPM, Accuracy tracking
- ❌ Mistake detection
- 📄 Random paragraphs
- 🔄 Auto next paragraph system
- 🏁 Final score at end only
- 📈 Progress bar
- 🎨 Modern UI design

---

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript

---

## ▶️ Run Project

Open `index.html` in browser or use VS Code Live Server.

---

## 📌 Note

Frontend-only project (no backend required).
